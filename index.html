Introducción

Este sitio web ha sido diseñado como parte de un proyecto académico, con el propósito de explorar las bases del diseño web utilizando HTML y CSS. Aquí podrás encontrar una muestra de cómo se estructuran las diferentes secciones de un sitio web funcional, incluyendo una introducción, una galería de imágenes y un formulario de contacto. Este proyecto demuestra cómo las tecnologías web básicas pueden ser utilizadas para crear páginas dinámicas y visualmente atractivas. Además, ofrece una experiencia intuitiva y amigable, con un diseño que se adapta a diversos dispositivos. ¡Espero que disfrutes explorando este espacio!
